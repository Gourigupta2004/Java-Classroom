## Components of Java 

1. Function 
> Here we writes what work needs to be executed
```
void main(){

}
```
> main is the function from where execution starts in JAVA

2. Class 
> we writes functions inside a class 
```
class Main{
    void main(){

    }
}
```
> Class is a big entity comprise of various functions
